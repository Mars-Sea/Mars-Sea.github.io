Windows 使用 Emby

2023-04-09 1 min read # Emby # 软件 # Windows

1. 下载客户端

下载地址：https://github.com/MediaBrowser/emby-theater-windows/releases

2. 解压下载的 Emby 软件

3. 在解压的文件夹中搜索 main.js，打开编辑

4. 在文件内搜索

https://tv.emby.media

5. 找到下面位置

function getAppBaseUrl() {

        var url = 'https://tv.emby.media';

        //url = 'http://localhost:8088';
        return url;
    }

6. 替换内容

function getAppBaseUrl() {
        var url = 'http://embycrack.tk';
        //url = 'http://localhost:8088';
        return url;
    }

7. 保存使用
